Descrição detalhada

Endpoint responsável por deletar um contrato.

Este endpoint implementa a interface IEndpoint e mapeia a rota de exclusão lógica de contratos. Espera receber o ID do contrato na rota, constrói uma requisição e invoca o handler para deletar o contrato. Se a exclusão for bem-sucedida, retorna NoContent; caso contrário, retorna BadRequest com os detalhes do erro.
