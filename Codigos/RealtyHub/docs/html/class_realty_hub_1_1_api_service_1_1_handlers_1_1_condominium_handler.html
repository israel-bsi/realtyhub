Descrição detalhada

Responsável pelas operações relacionadas a condomínios.

Esta classe implementa a interface ICondominiumHandler e fornece métodos para criar, atualizar, deletar e buscar condomínios no banco de dados.

Construtores e Destrutores

Este construtor é utilizado para injetar o contexto do banco de dados necessário para realizar operações CRUD nos condomínios.

inline

Retorna uma lista paginada de todos os condomínios ativos, com opção de filtro por busca.

Este método busca todos os condomínios ativos no banco de dados e aplica filtros de busca, se fornecidos.

A resposta é paginada com base nos parâmetros de paginação fornecidos na requisição e retorna uma lista de condomínios.

Caso o filtro não seja fornecido, retorna todos os condomínios ativos.


Parâmetros
requestRequisição que contém parâmetros de paginação e filtro.
Retorna
Retorna uma resposta paginada com os condomínios ativos com base no filtro ou um erro em caso de falha.
