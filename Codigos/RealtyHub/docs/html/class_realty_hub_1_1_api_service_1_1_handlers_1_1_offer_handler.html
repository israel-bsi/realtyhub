Descrição detalhada

Responsável pelas operações relacionadas a propostas.

Esta classe implementa a interface IOfferHandler e fornece métodos para criar, atualizar, rejeitar, aceitar e buscar propostas no banco de dados.

◆ UpdateAsync()

async Task< Response< Offer?> > RealtyHub.ApiService.Handlers.OfferHandler.UpdateAsync ( Offer request)
inline

Atualiza uma proposta existente no banco de dados.

Este método atualiza as informações de uma proposta existente no banco de dados.

O método verifica se a proposta existe e se o status da proposta permite a atualização.

Além disso, ele valida se o valor total dos pagamentos corresponde ao valor da proposta.

Se a proposta for encontrada e o status permitir, as informações da proposta são atualizadas.

Parâmetros
requestObjeto contendo as novas informações da proposta.
Retorna
Retorna a resposta com a proposta atualizada ou um erro se não for encontrada.
