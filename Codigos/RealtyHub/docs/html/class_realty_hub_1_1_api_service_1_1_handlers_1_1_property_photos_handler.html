Descrição detalhada

Responsável pelas operações relacionadas às fotos de imóveis.

Esta classe implementa a interface IPropertyPhotosHandler e fornece métodos para criar, atualizar, deletar e buscar fotos de imóveis no banco de dados.

Construtores e Destrutores

Este construtor é utilizado para injetar o contexto do banco de dados necessário para realizar operações CRUD nas fotos dos imóveis.
