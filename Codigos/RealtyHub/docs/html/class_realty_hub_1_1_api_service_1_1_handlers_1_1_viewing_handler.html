Descrição detalhada

Responsável pelas operações relacionadas às visitas de imóveis.

Esta classe implementa a interface IViewingHandler e fornece métodos para agendar, reagendar, finalizar e cancelar visitas, além de buscar visitas específicas ou todas as visitas.

◆ ScheduleAsync()

async Task< Response< Viewing?> > RealtyHub.ApiService.Handlers.ViewingHandler.ScheduleAsync ( Viewing request)
inline

Agenda uma nova visita para um imóvel.

Este método adiciona uma nova visita à base de dados e salva as alterações.

Ele verifica se o cliente e o imóvel existem e estão ativos antes de agendar a visita.

Se a visita já estiver agendada para a mesma data, retorna um erro.

O status da visita é definido como "Pendente" por padrão.

Parâmetros
requestObjeto contendo as informações para agendamento da visita.
Retorna
Retorna uma resposta com a visita agendada ou um erro em caso de falha.
