Descrição detalhada

Classe que representa uma requisição de registro de usuário.

Esta classe contém as propriedades necessárias para o registro de um novo usuário, incluindo o número do CRECI, nome, e-mail e senha.

As propriedades são validadas para garantir que os dados fornecidos estejam corretos e atendam aos requisitos mínimos.
