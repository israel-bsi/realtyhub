Descrição detalhada

Interface que define os métodos para manipulação de modelos de contrato.

Esta interface é responsável por definir as operações que podem ser realizadas com os modelos de contrato, como a recuperação de todos os modelos disponíveis.

Documentação das funções

◆ GetAllAsync()

Task< Response< List< ContractTemplate >?> > RealtyHub.Core.Handlers.IContractTemplateHandler.GetAllAsync ( )

Recupera todos os modelos de contrato disponíveis.

Este método é responsável por buscar todos os modelos de contrato armazenados no sistema e retornar uma lista com os resultados.

Retorna
Retorna um objeto Response<TData> contendo uma lista de modelos de contrato.

Implementado por RealtyHub.ApiService.Handlers.ContractTemplatesHandler e RealtyHub.Web.Handlers.ContractTemplateHandler.
