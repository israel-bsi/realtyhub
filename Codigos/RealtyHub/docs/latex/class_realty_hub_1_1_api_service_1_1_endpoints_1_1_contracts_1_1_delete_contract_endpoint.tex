\doxysubsection{Descrição detalhada}
Endpoint responsável por deletar um contrato. 

Este endpoint implementa a interface \doxylink{interface_realty_hub_1_1_api_service_1_1_common_1_1_api_1_1_i_endpoint}{IEndpoint} e mapeia a rota de exclusão lógica de contratos. Espera receber o ID do contrato na rota, constrói uma requisição e invoca o handler para deletar o contrato. Se a exclusão for bem-\/sucedida, retorna No\+Content; caso contrário, retorna Bad\+Request com os detalhes do erro.
