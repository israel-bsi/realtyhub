Atualiza uma proposta existente no banco de dados. 

Este método atualiza as informações de uma proposta existente no banco de dados.

O método verifica se a proposta existe e se o status da proposta permite a atualização.

Além disso, ele valida se o valor total dos pagamentos corresponde ao valor da proposta.

Se a proposta for encontrada e o status permitir, as informações da proposta são atualizadas.
