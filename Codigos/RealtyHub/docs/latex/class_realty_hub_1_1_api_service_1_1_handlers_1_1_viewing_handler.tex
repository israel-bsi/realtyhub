\doxysubsection{Descrição detalhada}
Responsável pelas operações relacionadas às visitas de imóveis. 

Esta classe implementa a interface \doxylink{interface_realty_hub_1_1_core_1_1_handlers_1_1_i_viewing_handler}{IViewing\+Handler} e fornece métodos para agendar, reagendar, finalizar e cancelar visitas, além de buscar visitas específicas ou todas as visitas.
