\doxysubsection{Descrição detalhada}
Classe que representa uma solicitação de recuperação de senha. 

Esta classe é usada para enviar uma solicitação de recuperação de senha para o sistema. Ela contém as informações necessárias para identificar o usuário que está solicitando a recuperação de senha.
