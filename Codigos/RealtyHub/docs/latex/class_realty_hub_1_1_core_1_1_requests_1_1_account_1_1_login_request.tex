\doxysubsection{Descrição detalhada}
Classe que representa uma solicitação de login de usuário. 

Esta classe é usada para enviar as credenciais de login do usuário para o sistema.

Ela contém as informações necessárias para autenticar o usuário e iniciar uma sessão.
