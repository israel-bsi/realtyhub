\doxysubsection{Descrição detalhada}
Interface que define os métodos para manipulação de contas de usuário. 

Esta interface é responsável por definir as operações que podem ser realizadas com as contas de usuário, como login, registro, confirmação de e-\/mail, recuperação de senha, etc.
