\doxysubsection{Descrição detalhada}
Interface que define os métodos para manipulação de visitas a imóveis. 

Esta interface é responsável por definir as operações que podem ser realizadas com as visitas de imóveis, como agendamento, reagendamento, conclusão e cancelamento.
